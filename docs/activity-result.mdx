# Activity Result

## Get Image from Camera
```kt
viewModelScope.launch(Dispatchers.Main) {
    val image = takePicturePreview()
}
```

## Get Content from storage/gallery

```kt
viewModelScope.launch(Dispatchers.Main) {
    val image = getContent("image/*")
}
```

## Intent Result(Start Activity for Result, etc)

```kt
viewModelScope.launch(Dispatchers.Main) {
    val result = intentResult(your_intent)
}
```
