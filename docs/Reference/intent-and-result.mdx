# Intent & Result

## Take picture preview

Get image **bitmap** from camera directly in `viewModel`.

```kt
val b = takePicturePreview()// suspend, get Bitmap?
```

## GetContent

Get content **image/document/pdf** etc directly in `viewModel`.

```kt
val b = getContent("image/*")// suspend, get Uri?
```

## StartActivityForResult

```kt
suspend fun intentResult(intent: Intent): ActivityResult?
```

## Custom

Make your own

```kt
suspend fun <I,O>requestResult(
    createIntent: (context: Context, input: I)->Intent,
    parseResult: (resultCode: Int, intent: Intent?)->O,
    launch: ManagedActivityResultLauncher<I,O>.()->Unit,
): O?
```
