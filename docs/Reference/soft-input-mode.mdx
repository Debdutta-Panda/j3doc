# SoftInputMode

Use `fun setSoftInputMode(mode: Int)` to change soft input mode.

Use any of the following value for mode:

1. `WindowManager.LayoutParams.SOFT_INPUT_ADJUST_PAN`
2. `WindowManager.LayoutParams.SOFT_INPUT_ADJUST_NOTHING`
3. `WindowManager.LayoutParams.SOFT_INPUT_ADJUST_UNSPECIFIED`
4. `WindowManager.LayoutParams.SOFT_INPUT_ADJUST_RESIZE`

Or the following from J3:

1. `SoftInputMode.adjustPan`
2. `SoftInputMode.adjustNothing`
3. `SoftInputMode.adjustUnspecified`
4. `SoftInputMode.adjustResize`
